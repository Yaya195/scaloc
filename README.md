# scaloc

Scalable indoor localization research codebase.

## Project Structure

This repository is scaffolded for:
- Data preprocessing and RP/graph construction
- Local and federated model training
- Evaluation and reproducible experimentation

## Quick Start

1. Create a Python environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Configure parameters in `configs/`.
4. Run scripts in `scripts/`.
